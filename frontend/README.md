# The Breath App

A minimal, elegant breathing practice application built with React, Vite, and Node.js.

## Features
- **Breathing Techniques**: Multiple patterns including Box Breathing, 4-7-8, and Deep Belly Breathing.
- **Mobile Friendly**: Fully responsive design that transforms into a bottom navbar on mobile.
- **Progress Tracking**: Local history of your breathing sessions.
- **Customizable**: Multiple themes and adjustable durations.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Lucide Icons, Vite.
- **Backend**: Node.js, Express, MongoDB.

## Repository
Find the source code here: [https://github.com/vishwascr/breathingapp](https://github.com/vishwascr/breathingapp)

## Development
1. Clone the repository.
2. Install dependencies in both `frontend` and `backend`.
3. Start the backend: `node server.js` (port 5035).
4. Start the frontend: `npm run dev` (port 3002).
